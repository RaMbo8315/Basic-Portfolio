About Me

Hello, World! My name is Henry Urena. I was born in Queens, New York and my family moved to Orlando, FL when I was 5 yrs old. I went to Dr. Phillips High School. I'm currently self employed as an antique car detailer. I started UCFs coding boot camp to make a career change. I'm pretty new to web development and I'm enjoying every bit of it as I go through UCFs intense boot camp. I gained an interest in web development when I took a class for it at valencia as an elective. We learned basic html and css but we never touched on javascript. I am looking forward to building great looking responsive web pages.
